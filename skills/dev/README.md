# dev — 开发质量 Skills

提升代码质量、调试效率和测试覆盖的 Skills。适用于所有软件开发项目。

## 包含 Skills（6 个）

| 名称 | 用途 | 触发时机 |
|------|------|---------|
| **express-typescript-api** | Express + TypeScript REST API 规范 | 构建/审查后端路由、中间件、认证、DB 操作时 |
| **systematic-debugging** | 系统化排查 bug 根因 | 遇到 bug、测试失败、意外行为时 |
| **test-driven-development** | 测试驱动开发 | 实现功能或修复 bug 前 |
| **verification-before-completion** | 声称完成前自动验证 | 准备提交或声称完成时 |
| **webapp-testing** | Playwright 浏览器自动化测试 | 需要验证前端功能、调试 UI 行为时 |
| **react-native-expo** | Expo React Native TypeScript 开发规范 | 构建/审查 Expo 移动端代码时 |

## 推荐组合

- **最小配置**：systematic-debugging + verification-before-completion
- **完整配置**：全部 6 个
- **纯后端项目**：express-typescript-api + systematic-debugging + test-driven-development
- **移动端全栈**：express-typescript-api + react-native-expo + 其余质量 Skills
